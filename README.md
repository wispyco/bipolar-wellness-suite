# Bipolar Wellness Suite

A progressive collection of 12 research-backed mini web apps for bipolar disorder self-management and research exploration. Each app builds on the previous ones, forming a comprehensive wellness toolkit.

Built over 12 hours — one app per hour — grounded in evidence-based psychotherapies including CBT, IPSRT, chronotherapy, psychoeducation, and functional remediation.
