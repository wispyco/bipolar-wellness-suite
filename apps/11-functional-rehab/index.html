Functional Remediation

Cognitive training exercises and occupational goal tracking based on functional remediation research by Torrent et al. Targets neurocognitive deficits that persist between mood episodes.

Attention

Number Sequence

Watch the numbers flash and repeat the sequence. Tests sustained attention and working memory.

3-5 minBeginner

Color-Word Challenge

Name the color of the text, not the word. Based on the Stroop test for selective attention.

2-4 minIntermediate

Memory

Memory Match

Find matching pairs of cards. Trains visual memory and pattern recognition.

3-5 minBeginner

Executive Function

Category Sort

Sort items into the correct categories as quickly as possible. Builds cognitive flexibility.

3-5 minIntermediate

Occupational Goals

Set specific, measurable goals for work, school, or daily responsibilities. Break big goals into small steps.
